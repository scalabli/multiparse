`Forever Scalable`

**Multiparse** is a nifty streaming multiple parser for python.


Multiparse requires Python `3.6.1` or later

## Getting Started

### Installation
You can install multiparse via the Python Package Index (PyPI)

```
pip install -U multiparse
```
